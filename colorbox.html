Colorbox

Enlace para abrir una página html en colorbox.

Responsividad

Para abrir una ventana modal con un HTML solo hace falta añadir la clase cboxElementIframe al enlace. Abrirá la página indicada en el href en una ventana modal.

Es necesario añadir tanto los estilos como la librería JS de Colorbox.

Las ventanas emergentes de colorbox se adaptan al tamaño de la ventana en que se abren.

Código

						
<link href="css/colorbox-galeria.css" rel="stylesheet">
<script src="js/jquery.colorbox-min.js"></script>
<a href="[ENLACE]" class="cboxElementIframe">[TEXTO_ENLACE]</a>
